进一步配置 打开目录
iView Cli 已将所有配置文件保存在指定的目录，按照下面的流程可以启动项目。对于一些定制内容，可以进一步配置。 请预先全局安装 Node.js、npm 、webpack。

安装依赖

$ npm install

启动服务（开发环境）

$ npm run dev

服务启动后，打开 http://127.0.0.1:8080 访问网站

编译打包（生产环境）

$ npm run build

修改 html 模板

修改 src/template/index.ejs 文件，使用 webpack 重新编译，可以改变入口 html。

返回首页 打开目录
